# Сайт Анны Минаковой

Одностраничный сайт-лендинг (think tank / strategic advisory). Тёмный космический стиль, золото точечно, шрифты Playfair Display + Manrope.

## Структура

```
site/
├── index.html        — сам сайт (всё медиа вынесено в assets/)
├── assets/
│   ├── planet.mp4         — видео-планета в hero
│   ├── planet-poster.jpg  — постер видео (пока грузится)
│   ├── anna.png           — фото Анны в hero
│   ├── anna-life-1.jpg    — фото в блоке ANNA LIFE
│   ├── nebula.png         — туманность под блоком «О себе»
│   └── cta-space.jpg      — космос-фон у формы
└── README.md
```

## Публикация на GitHub Pages

1. Создать (или открыть) репозиторий, напр. `AnnaMinakova`.
2. Залить **содержимое папки `site/`** в корень ветки (чтобы `index.html` лежал в корне).
3. Settings → Pages → Source: ветка `main`, папка `/ (root)` → Save.
4. Сайт будет доступен по адресу `https://<username>.github.io/<repo>/`.

> Важно: `index.html` ссылается на медиа относительными путями `assets/...`,
> поэтому папку `assets/` нужно залить рядом с `index.html`.

## Кастомный домен (опционально)

Settings → Pages → Custom domain: `minakovanna.ru`. У регистратора домена добавить
CNAME/A-записи на GitHub Pages (по их инструкции). В репозиторий добавить файл `CNAME`
с одной строкой: `minakovanna.ru`.

## Как продолжать редактировать с помощью Claude

Вариант А (через GitHub-коннектор Claude): подключить GitHub в Claude, дать доступ к репозиторию —
тогда Claude сможет читать `index.html`, вносить правки и готовить коммит/PR прямо в репозитории.

Вариант Б (вручную): скачать `index.html` из репозитория, прикрепить его в чат Claude,
попросить правку, получить обновлённый файл и закоммитить обратно.

## Технические заметки

- Дизайн-система в `:root` (цвета, шрифты). Настройки hero-видео — там же (`--hero-video-*`).
- Документы (политики, согласия) вшиты и открываются в модалках.
- Адаптив 360–1920px; анимации на чистом CSS/JS.
